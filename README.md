# Detailed Resume Builder using Streamlit

## Description
This project is a professional resume builder created using the Streamlit module in Python using streamlit. It allows users to generate a polished and customized resume with a sidebar that includes a profile picture and interactive icons linking to social media accounts. The project also features animated elements to enhance the visual appeal of the resume. If you want to add more features, absolutely you can. This is Just a basic structure for you. 
You just need to edit the relevant details, and make it your own.

## Installation
1. Clone the repository:
   using gitbash type
   git clone https://github.com/yourusername/resume-builder
   
2. Navigate to the project directory:
   using gitbash type
   cd resume-builder
   
3. Install the required dependencies:
   using gitbash type
   pip install -r requirements.txt
   
4. Run the application:
   using gitbash type
   streamlit run app.py
   

## Usage
- Customize the resume template by editing the `app.py` file.
- Add your personal information and adjust the layout as needed.
- Click the icons in the sidebar to link to your social media profiles.
